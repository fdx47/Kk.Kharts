# KropKontrol IoT Dashboard

Bienvenue dans le projet **KropKontrol IoT Dashboard**, une plateforme dédiée à la gestion et à la visualisation des données issues de capteurs pour l’agriculture intelligente et connectée 4.0.

Ce tableau de bord a été conçu pour offrir une interface intuitive et performante permettant de suivre en temps réel les conditions agricoles grâce à des capteurs IoT innovants. Notre objectif est de faciliter la prise de décision, optimiser les ressources et améliorer la productivité tout en respectant l’environnement.

## Installation

```bash
npm install
```

## Génération du build

### Production

```bash
npm run build
##!!!!!!!!!!!!!!!!!!!ne pas remplacer le fichier .htaccess présent à la racine du FTP!!!!!!!!!!!!!!!!!
```

### Environnement FTP de développement

L'archive générée est destinée à être copiée sur votre serveur dans le dossier `/vue`.

```bash
npm run build:devftp
```
